Loader

loader.open(engine='segyio', endian='big', strict=False, ignore_geometry=True)

Selector class for loading SEG-Y with either segyio-based loader or memmap-based one.

Parameters:

  • path (str) – Path to the SEG-Y file

  • engine ('memmap' or 'segyio') – Engine to load data from file: 'memmap' is based on numpy.memmap created for the whole file and 'segyio' is for using segyio library instruments. in any case, segyio is used to load information about the entire file (e.g. 'sample_interval' or 'shape').

  • endian ('big' or 'little') – Byte order in the file.

  • strict (bool) – See segyio.open()

  • ignore_geometry (bool) – See segyio.open()

Return type:

MemmapLoader or SegyioLoader
